# anubot-backend
## 소개
아누봇의 백엔드 API 서버입니다.

## API 문서
https://anubot.azurewebsites.net

## 코딩 표준
* DateTime 포맷은 UTC 시간대를 사용합니다.
* secret(ex. API키, DB 연결 정보)은 git에 저장하지 않습니다.
